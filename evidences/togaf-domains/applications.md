# Applications Domain Evidence

## Public summary

The article describes an application landscape with a core banking component, client and advisor channels, API integration layers and corporate support platforms.

## Public categories

| Category | Public description |
| --- | --- |
| Core banking | Legacy transactional system category. |
| Client channel | Mobile or digital channel for clients. |
| Advisor channel | Field channel for business advisors. |
| API integration | Gateway and integration components. |
| Corporate support | Administrative, reporting and productivity systems. |

## Public gap

The article identifies architectural duality between a monolithic core and microservice-oriented channels, with heterogeneous integration layers. This repository intentionally avoids real API contracts, internal endpoints and business logic.
