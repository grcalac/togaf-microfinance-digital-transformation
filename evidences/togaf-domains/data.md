# Data Domain Evidence

## Public summary

The data domain identifies three generic information entities: credits, clients and savings. The article also states that the transactional database remains on-premise because of regulatory constraints.

## Public data entities

| Entity | Public description |
| --- | --- |
| Credits | Data related to credit lifecycle management. |
| Clients | Data related to customer relationship and identification. |
| Savings | Data related to savings products and account activity. |

## Public gap

The AS-IS diagnosis identifies the need for stronger traceability of access to sensitive data. This repository does not include data samples, database schemas or real access logs.
