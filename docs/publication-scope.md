# Publication Scope

## Public-by-default rule

This repository assumes that every file is public. For that reason, no document should contain restricted, confidential or operationally sensitive information.

## Allowed content

- Summaries derived from the article.
- Anonymized evidence catalogs.
- Conceptual diagrams.
- Synthetic examples.
- Terraform reference modules using fictitious names and placeholders.
- Security, governance and ethical publication checklists.

## Prohibited content

- Source code of microservices.
- Real business rules or internal logic.
- Customer, employee or transaction data.
- Logs, screenshots or exports from real systems.
- IP addresses, internal DNS names, ARNs, account IDs, keys or tokens.
- Real VPN configuration.
- Real CI/CD credentials.
- Exact production topology.
- Non-public regulatory evidence.

## Reviewer note

The absence of operational details is intentional. It protects the institution and keeps the repository aligned with responsible academic disclosure.
