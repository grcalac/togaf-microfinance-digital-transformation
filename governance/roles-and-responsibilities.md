# Roles and Responsibilities

These roles are conceptual and support publication governance for the repository.

| Role | Responsibility |
| --- | --- |
| Article author | Maintains methodological consistency with the article. |
| Evidence curator | Ensures evidence is public-safe and traceable. |
| Architecture reviewer | Reviews architecture descriptions for accuracy and safe disclosure. |
| Security reviewer | Checks for secrets, sensitive infrastructure details and disclosure risk. |
| Academic reviewer | Uses the repository to verify methodological transparency. |

## Public access

All repository content is public. There are no private folders, private branches or restricted-access evidence files in this repository.
